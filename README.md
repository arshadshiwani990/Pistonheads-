PistonHeads Car Ads Scraper
This project is a web scraper built using Scrapy, a Python framework, to extract car advertisement data from the website PistonHeads. The goal of this scraper is to gather relevant information about car listings, including contact details, to facilitate purchasing cars from sellers.

Features
Efficient Data Extraction: Automatically scrapes car ad details such as titles, prices, locations, and contact information.
Customizable Filters: Allows filtering of ads based on car make, model, price range, and more.
Export to CSV: Saves scraped data into a CSV file for easy analysis and future reference.
Contact Information: Extracts available contact details of the car owners from the listings.
